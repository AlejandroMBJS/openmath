# Sistemas Coordenados

## Punto de partida

Un sistema coordenado es un mecanismo para describir puntos mediante n-tuplas. No es la geometria misma, sino una interfaz algebraica que permite medir, calcular y comparar magnitudes.

## Sistema de referencia

En el caso euclideo clasico, un sistema coordenado se fija eligiendo:

- un origen;
- una base del espacio vectorial asociado;
- una orientacion y, si hace falta, una escala.

Con ello cada punto queda descrito por sus componentes respecto de esa base.

## Coordenadas cartesianas

En $\mathbb{R}^3$, las coordenadas cartesianas escriben un punto como

$$
(x,y,z).
$$

La base canónica es ortonormal y las distancias se expresan mediante la forma cuadrática euclidea

$$
ds^2=dx^2+dy^2+dz^2.
$$

## Por que cambiar de coordenadas

Cambiar de coordenadas no altera el fenómeno físico ni el objeto geométrico. Cambia solo el lenguaje de descripción. La elección correcta puede convertir una ecuación complicada en una separación natural de variables.

## Sistemas adaptados

Los sistemas coordenados más útiles suelen estar adaptados a simetrías:

- cartesiano para simetría traslacional;
- polar o cilíndrico para simetría axial;
- esférico para simetría radial.

Esta idea es central en física matemática.

## Coordenadas y observables

Al cambiar de coordenadas, las componentes de un vector o tensor cambian, pero el objeto geométrico subyacente permanece. La distinción entre objeto y componentes es una de las lecciones más importantes de este bloque.

## Cierre

Un sistema coordenado es una elección estructurada de descripción. Su valor no está en ser único, sino en adaptarse al problema y permitir una traducción eficaz entre geometría, álgebra y análisis.

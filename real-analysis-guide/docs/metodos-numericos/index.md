# Metodos Numericos

## Objetivo del bloque

Este bloque organiza el estudio de aproximacion numerica de raices, derivadas, integrales, ecuaciones diferenciales y sistemas lineales.

El bloque ya arranca con paginas reales para metodos de busqueda de raices. El resto del temario permanece listado en el mapa para seguir extendiendolo con el mismo criterio.
